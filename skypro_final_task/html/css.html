    Атрибуты

  • class — С помощью него добавляют CSS-классы элементам.

    Свойства

  • font-size — Размер шрифта.
  • background — Фон страницы.

    Значения

  • xx-small — Самый маленький.
  • medium — Средний.
  • x-large — Самый большой.
